Sobre mí

Soy informático y desarrollador de software con foco en el desarrollo backend con Python y Django, pero con experiencia full stack. Me gusta diseñar soluciones simples pero escalables, escribir código limpio y mantenible, y trabajar usando buenas prácticas de control de versiones y despliegue.

Tengo experiencia construyendo aplicaciones web completas, desde el levantamiento de requerimientos hasta la puesta en producción. Uno de mis proyectos más relevantes es un sistema de postulaciones laborales, con backend en Python, Django, MySQL y APIs REST, y un frontend desarrollado en HTML5, CSS, Bootstrap, JavaScript y jQuery.

Me interesa especialmente la automatización de procesos, el diseño de APIs REST y la integración de sistemas. Disfruto liderar proyectos, trabajar bajo presión cuando es necesario y seguir aprendiendo nuevas tecnologías de forma constante.

Lo que puedo aportar

  • Desarrollo de aplicaciones web y APIs REST con Python, Django y DRF.
  • Diseño de bases de datos relacionales e integración con MySQL/PostgreSQL.
  • Despliegues en la nube (por ejemplo, PythonAnywhere) y automatización básica.
  • Gestión de código con Git, GitHub y GitLab, trabajo colaborativo y revisión de código.
  • Liderazgo técnico en proyectos de principio a fin y comunicación clara con el cliente.
